# Mobile Banking Application

## Overview

The Mobile Banking Application is a secure and scalable banking platform that enables customers to perform essential banking operations through a mobile application. The system allows users to register, securely log in, manage bank accounts, transfer funds, pay utility bills, view transaction history, and receive real-time notifications.

This project follows modern Software Engineering, Agile, and DevOps practices. The application is developed using a three-tier architecture consisting of a Flutter frontend, a Spring Boot backend, and a MySQL database. The project also incorporates Git, Docker, Jenkins, AWS, Prometheus, and Grafana to automate software delivery and monitor the deployed application.

## System Architecture

The Mobile Banking Application follows a three-tier architecture.

* **Presentation Layer:** Flutter Mobile Application
* **Application Layer:** Spring Boot REST APIs, Business Logic, Authentication & Authorization
* **Data Layer:** MySQL Database

The DevOps pipeline automates software integration, testing, deployment, and monitoring using Git, Jenkins, Docker, AWS EC2, Prometheus, and Grafana.

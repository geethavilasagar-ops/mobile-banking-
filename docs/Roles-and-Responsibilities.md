# Roles and Responsibilities

# Mobile Banking Application

## Project Team Structure

The Mobile Banking Application is developed using Agile Software Development practices. Each team member has clearly defined responsibilities to ensure smooth project execution.

---

# 1. Product Owner

### Responsibilities

* Gather project requirements.
* Define project objectives.
* Prioritize product backlog.
* Approve completed features.
* Coordinate with stakeholders.

Deliverables

* Requirements Document
* Product Backlog
* Feature Approval

---

# 2. Scrum Master

### Responsibilities

* Plan sprint activities.
* Conduct sprint meetings.
* Track project progress.
* Remove development blockers.
* Ensure Agile practices are followed.

Deliverables

* Sprint Plan
* Sprint Review
* Sprint Retrospective

---

# 3. Frontend Developer

### Responsibilities

* Develop the Flutter mobile application.
* Design responsive user interfaces.
* Integrate REST APIs.
* Validate user inputs.
* Improve user experience.

Deliverables

* Flutter Application
* UI Screens
* API Integration

---

# 4. Backend Developer

### Responsibilities

* Develop REST APIs using Spring Boot.
* Implement business logic.
* Connect the application to MySQL.
* Handle authentication and authorization.
* Maintain application security.

Deliverables

* REST APIs
* Business Logic
* Database Integration

---

# 5. Database Designer

### Responsibilities

* Design the database schema.
* Create ER diagrams.
* Define relationships.
* Optimize database queries.
* Maintain data integrity.

Deliverables

* Database Schema
* ER Diagram
* SQL Scripts

---

# 6. QA / Test Engineer

### Responsibilities

* Prepare test cases.
* Perform functional testing.
* Report software defects.
* Verify bug fixes.
* Ensure software quality.

Deliverables

* Test Cases
* Bug Reports
* Test Reports

---

# 7. DevOps Engineer

### Responsibilities

* Manage Git and GitHub repositories.
* Configure Docker containers.
* Build Jenkins CI/CD pipelines.
* Deploy applications to AWS EC2.
* Configure Prometheus and Grafana.
* Automate build, test, and deployment processes.
* Monitor application health and performance.
* Maintain deployment documentation.

Deliverables

* Docker Configuration
* Jenkins Pipeline
* Deployment Scripts
* Monitoring Dashboards
* CI/CD Automation

---
